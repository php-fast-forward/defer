Getting Started
===============

Welcome to **FastForward Defer**!

FastForward Defer brings Go-style `defer` semantics to PHP, enabling deterministic, scope-bound cleanup and resource management. This documentation will guide you through installation, core concepts, usage patterns, advanced features, and API reference.
