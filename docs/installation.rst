Installation
============

FastForward Defer requires **PHP 8.3+** and Composer.

.. code-block:: bash

   composer require php-fast-forward/defer

No additional configuration is required. The package is PSR-4 autoloaded and ready for use.

**Requirements:**

- PHP 8.3 or newer
- Composer

**Optional Integrations:**

- PSR-3 Logger (for advanced error reporting)
- PSR-14 Event Dispatcher (for event-driven error handling)
- PSR-15 HTTP Middleware (for request lifecycle management)
